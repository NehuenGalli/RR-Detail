# RR Detail - Landing Page Automotriz 🚗✨

Una landing page moderna, responsiva y optimizada para un taller de estética vehicular (Detailing). Este proyecto fue desarrollado para mostrar servicios, trabajos realizados en formato de galería dinámica, y facilitar el contacto directo con los clientes vía WhatsApp.

🔗 **[Visitar el sitio web en vivo](https://rrdetail.com.ar/)**

## 🚀 Características Principales

* **Diseño 100% Responsivo:** Adaptable a dispositivos móviles, tablets y pantallas de escritorio.
* **Galería de Trabajos (Sliders):** Carruseles de imágenes independientes creados con Vanilla JavaScript puro (sin librerías externas) para mostrar el antes y el después de los vehículos.
* **Animaciones de Interfaz:** Efectos de aparición en cascada (Hero) y animaciones de scroll (Intersection Observer) para una experiencia de usuario más fluida.
* **Sección de Preguntas Frecuentes (FAQ):** Sistema de acordeón interactivo para mejorar la experiencia del usuario sin sobrecargar visualmente la página.
* **Integración con WhatsApp:** Botón flotante optimizado con un mensaje predefinido para agilizar la captación de clientes.
* **Performance y SEO:** Implementación de carga diferida de imágenes (`lazy loading`), formatos modernos (`.webp`), precarga de recursos (preload) y etiquetas meta optimizadas para buscadores.

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructura semántica y accesibilidad.
* **CSS3:** Flexbox, variables CSS (Custom Properties), animaciones personalizadas (Keyframes) y Media Queries.
* **JavaScript (Vanilla):** Lógica de manipulación del DOM para sliders múltiples e interacciones de la interfaz.
* **Despliegue:** Vercel (con dominio personalizado).
